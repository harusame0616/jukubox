# カテゴリー一覧

JukuBox のコースカテゴリー一覧。`packages/supabase/categories.ts` の `categoryTree` をシード（`seed-categories.ts`）で `categories` テーブルへ投入している。

- パスは ltree 形式でユニーク（現状は 1 階層なので slug がそのまま path）
- `category_id` は `path` から `uuidv5`（namespace: `a0a0a0a0-1111-4111-8111-000000000002`）で決定論的に生成
- 表記は `slug` / `name`

## 設計方針

- **カテゴリーは 1 階層の大分類のみ**。具体技術（Next.js / Tailwind CSS / Claude Code 等）は `courses.tags`（JSONB）で自由に付与する
- `courses.category_id` は単一の大分類 FK、`courses.tags` は複数の具体技術タグ、と役割を分離
- これによりカテゴリーとタグの責務が衝突せず、コース投稿者は「どの大分類か」だけ決めれば良い
