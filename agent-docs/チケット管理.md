# チケット管理

## ツール

ClickUp

## インターフェイス

Web API (curl 経由)

## リスト情報

リスト ID： 901817311656

## 認証

環境変数の `CLICKUP_API_KEY` に API KEY を設定
`CLICKUP_API_KEY` が設定されているか中身の確認は絶対に行わず、設定済みの前提でリクエストを行う
認証エラーが返ってきた場合は、`~/.claude/settings.json` の `env` に `CLICKUP_API_KEY` が正しく設定されているかユーザーに確認を促す

## チケットの取得

- リッチコンテンツが取得できるように include_markdown_description=true を付与する

## チケットの作成・更新

### テンプレート

チケットの説明は以下のテンプレートに従う
各セクションは必ず用意するが、記載内容がない場合は「なし」と記載する

```
## 概要

<簡潔な概要>

## 詳細

<チケットの背景や詳細な説明>

## やること

- [ ] <タスクを分解してチェックボックスで記載>

## やらないこと

- [ ] <今回のチケットでは対応しないことをチェックボックスで記載>

## 設計

<具体的な設計や実装方針など。特に検討したが採用しなかった方法や採用した方法の理由なども記載すること>

## 参考情報

<SLACK やドキュメントなど関連情報へのリンク>

```

### 注意事項

- チケットの説明を更新する際は `markdown_description` フィールドを使用する
